### 核心思想

> scDrugMap 是一个将单细胞扰动数据映射到药物反应预测的统一框架。它利用单细胞转录组学的分辨率来预测药物在细胞类型级别的效应，为药物发现和精准医学提供了新的工具。

## 2. 模型架构

### 2.1 框架流程

1. 收集单细胞扰动数据
2. 建立扰动-药物映射关系
3. 预测药物在各类细胞中的效果

### 2.2 应用场景

- 药物重定位
- 细胞类型特异性药物效应
- 组合药物预测

## 3. 核心创新

### 3.1 核心贡献

- 连接单细胞扰动与药物响应
- 细胞类型级别分辨率
- 支持药物发现和精准医学
