## 2. 模型架构

### 2.1 主要结论

"PCA still rules" 发现：**在扰动预测任务中，简单的 PCA + 线性回归基线方法并不比复杂的深度学习模型差**，甚至在某些场景下更好。

### 2.2 实验设计

- **对比方法**: PCA+线性回归 vs. scGPT、Geneformer、scFoundation 等 SOTA 模型
- **任务**: 预测基因扰动后的表达变化
- **数据集**: 多个 Perturb-seq 数据集

## 3. 核心创新

### 3.1 对单细胞基础模型的系统性质疑

揭示了许多被广泛引用的 SOTA 结果实际上来自不恰当的评估设置。

### 3.2 重要启示

- 复杂的深度学习模型并非总是必要
- 评估基线（如 PCA）应该被更认真地对待
- 该领域的进步需要更严格的基准测试
