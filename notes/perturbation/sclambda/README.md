# scLAMBDA 学习笔记

> scLAMBDA 是一个基于深度生成模型的框架，用于建模和预测单细胞多基因扰动响应。它利用大语言模型的嵌入能力来编码基因和扰动信息，并通过解耦表示学习分离基线细胞状态和扰动状态表征，从而实现对未观测靶基因或基因组合的扰动结果预测。支持单基因和组合式多基因扰动的 in silico 预测。

### 核心思想

> scLAMBDA 是一个基于深度生成模型的框架，用于建模和预测单细胞多基因扰动响应。它利用大语言模型的嵌入能力来编码基因和扰动信息，并通过解耦表示学习分离基线细胞状态和扰动状态表征，从而实现对未观测靶基因或基因组合的扰动结果预测。支持单基因和组合式多基因扰动的 in silico 预测。

## 2. 模型架构

### 2.1 核心组件

- **LLM 基因嵌入**: 利用 LLM 生成基因/蛋白的功能描述嵌入
- **解耦 VAE**: 分离基线细胞表征与扰动状态表征
- **扰动预测器**: 对未观测扰动进行 in silico 预测

### 2.2 解耦表示学习

```
basal_z, perturbation_z = encoder(expression, perturbation)
reconstructed = decoder(basal_z + perturbation_z)
```

## 3. 核心创新

### 3.1 多基因扰动

支持单基因和组合式多基因扰动预测，利用 LLM 知识泛化到未观测组合。

### 3.2 In Silico 预测

- 预测未观测靶基因的扰动效果
- 预测未观测基因组合的效果
- 单细胞级别的预测分辨率
