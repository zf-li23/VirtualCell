# CONCH 学习笔记

> CONCH（CONtrastive learning from Captions for Histopathology）是一个视觉-语言基础模型，用于计算病理学。在 117 万张图像-描述配对的最大病理学视觉-语言数据集上通过任务无关预训练训练。在 14 个不同基准上评估，支持图像分类、分割、描述生成、文本到图像和图像到文本检索等任务。不使用 TCGA/PAIP/GTEX 等公开数据集进行预训练，避免数据污染风险。

### 核心思想

> CONCH（CONtrastive learning from Captions for Histopathology）是一个视觉-语言基础模型，用于计算病理学。在 117 万张图像-描述配对的最大病理学视觉-语言数据集上通过任务无关预训练训练。在 14 个不同基准上评估，支持图像分类、分割、描述生成、文本到图像和图像到文本检索等任务。不使用 TCGA/PAIP/GTEX 等公开数据集进行预训练，避免数据污染风险。

## 2. 模型架构

### 2.1 对比学习框架

CONCH 采用 CLIP 风格的视觉-语言对比学习：

1. **图像编码器**: Vision Transformer (ViT)
2. **文本编码器**: BERT 风格的 Transformer
3. **对比目标**: 最大化匹配图像-文本对的余弦相似度

### 2.2 预训练数据

117 万张病理图像-描述对，来自多种来源（组织类型、染色方式、放大倍数各异）。特别地，未使用 TCGA、PAIP、GTEX 等常见的公开数据集。

## 3. 核心创新

### 3.1 病理领域最大 VL 数据集

117 万对，是目前病理领域最大的视觉-语言数据集。

### 3.2 广泛的下游迁移

14 个基准任务上的 SOTA 表现，远超其他病理视觉-语言模型。

### 3.3 非 H&E 泛化

对 IHC 和特殊染色也有优异的表示能力。

### 3.4 数据无污染

不使用 TCGA/PAIP/GTEX 预训练，降低基准数据污染风险。
