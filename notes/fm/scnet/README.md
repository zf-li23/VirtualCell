# scNET 学习笔记

> scNET 通过将 scRNA-seq 数据与蛋白质-蛋白质相互作用（PPI）网络集成，利用双图（bigraph）架构同时学习上下文特异的基因嵌入和细胞嵌入。它用 GNN 建模基因-基因关系（基于 PPI），用注意力机制建模细胞-细胞关系，解决了纯表达数据信噪比低、无法捕获通路级变化的问题。

### 核心思想

> scNET 通过将 scRNA-seq 数据与蛋白质-蛋白质相互作用（PPI）网络集成，利用双图（bigraph）架构同时学习上下文特异的基因嵌入和细胞嵌入。它用 GNN 建模基因-基因关系（基于 PPI），用注意力机制建模细胞-细胞关系，解决了纯表达数据信噪比低、无法捕获通路级变化的问题。

## 2. 模型架构


### 2.1 双图架构（Bigraph）

```text
基因表达数据            PPI 网络 (STRING)
    │                      │
    ▼                      ▼
基因级别编码 ──── 基因图 (GNN) ──── 基因嵌入
    │
    │  基于基因嵌入构建
    ▼
细胞图 (注意力) ──── 细胞嵌入
```

### 2.2 两个关键组件

- **基因图 (Gene Graph)**: 基于 STRING PPI 网络，基因作为节点，PPI 作为边，用 GNN 传播信息
- **细胞图 (Cell Graph)**: 基于基因嵌入的相似性构建，用注意力机制建模细胞关系

### 2.3 去噪能力

通过 PPI 先验的约束，模型可以有效过滤表达数据中的噪声，聚焦于有生物学功能的基因模块。



## 3. 核心创新


### 3.1 PPI + 表达的双模态融合

首个将蛋白质-蛋白质相互作用网络作为先验知识引入单细胞表示学习的框架。

### 3.2 上下文特异的基因嵌入

同一基因在不同细胞类型/状态下可以有不同的嵌入——反映了其功能的上下文依赖性。

### 3.3 与同类模型对比

| 维度 | scNET | scGPT | Geneformer |
|------|-------|-------|-----------|
| 先验知识 | **PPI 网络** | 无 | 无 |
| 基因嵌入 | 上下文特异 | 固定 | 固定 |
| 架构 | GNN + 注意力 | Transformer | Transformer |
| 去噪能力 | **强** | 中等 | 中等 |

## 9. 关键概念 Q&A


**Q: PPI 网络在 scNET 中的作用？**
A: PPI 网络提供了基因功能关系的先验知识——两个基因如果有 PPI 连接，它们的嵌入应该相互接近。这帮助模型在噪声较大的表达数据中找到有意义的基因模块。

**Q: scNET 可以用于任何物种吗？**
A: 目前主要支持人类（使用 STRING 人类 PPI 网络），但理论上可以扩展到其他有 PPI 数据的物种。
