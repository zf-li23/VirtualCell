# CellFM 学习笔记

> CellFM 是腾讯 AI Lab 开发的**大规模单细胞基础模型**，在 **1 亿人类细胞**上预训练。作为目前公开报道中预训练数据量最大的单细胞基础模型之一，CellFM 专注于通过海量多样本数据学习通用的细胞表示，在下游多种任务中展现了优异性能。

### 核心思想

> **"数据量就是王道"**——通过汇聚 1 亿细胞的超大规模预训练数据，让模型学习到足够通用的细胞表示，从而在各类下游任务中表现优异。

## 2. 模型架构

标准 Transformer Encoder 架构（BERT 风格），使用 [CLS] token 的输出作为细胞嵌入。

## 3. 核心创新

### 3.1 1 亿细胞预训练

CellFM 的最大特色是预训练数据规模——1 亿人类细胞，是目前最大的单细胞预训练数据集之一。

### 3.2 与同类模型的对比

| 维度 | CellFM | scFoundation | scGPT | Geneformer |
|------|--------|-------------|-------|-----------|
| **预训练细胞** | **1 亿** | 5000 万 | 3300 万 | 3000 万 |
| **架构** | BERT | 非对称 AE | GPT | BERT |
| **参数** | 未公开 | 3B | ~50M | 6.5M |

## 4. 数据预处理

标准流程：对数归一化 → 高变基因选择 → 基因表达值编码。

## 5. Tokenization 与输入编码

### 5.1 基因编码
使用 Ensembl Gene ID 作为 token，通过可学习嵌入表映射为向量。

### 5.2 表达值编码
使用连续值编码（线性层将标量映射到向量空间）。

## 6. 预训练

### 6.1 预训练数据

### 6.2 预训练目标
掩码基因表达预测（MSE 损失）。
