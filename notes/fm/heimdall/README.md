### 核心思想

> Heimdall 是一个针对单细胞基础模型的通用词元化（tokenization）框架。它专注于解决单细胞数据中基因表达的离散化和标记化问题，提出了更高效、更具生物学意义的词元化策略，旨在为各种单细胞基础模型提供统一的输入处理接口。

## 2. 模型架构

### 2.1 Tokenization 策略

Heimdall 提供了多种基因表达词元化策略：
- **表达水平分箱**: 将表达值离散化为词元
- **基因-表达联合编码**: 将基因身份和表达水平联合编码
- **上下文感知词元化**: 考虑局部表达模式的词元化

### 2.2 设计目标

- 保留生物学信息的同时最大化压缩率
- 与多种基础模型架构兼容
- 提供标准化的输入处理接口

## 3. 核心创新

### 3.1 核心贡献

- 统一单细胞基础模型的输入处理
- 系统比较不同词元化策略的影响
- 高效处理大规模 scRNA-seq 数据
