# CellPLM 学习笔记

> CellPLM 是首个将预训练从'单细胞级别'推广到'细胞-细胞关系级别'的工作。不同于 Geneformer/scGPT 等只编码基因关系的模型，CellPLM 显式建模细胞-细胞相似性图，在保持百倍推理速度优势的同时，在多种下游任务中一致超越现有预训练模型。

### 核心思想

> CellPLM 是首个将预训练从'单细胞级别'推广到'细胞-细胞关系级别'的工作。不同于 Geneformer/scGPT 等只编码基因关系的模型，CellPLM 显式建模细胞-细胞相似性图，在保持百倍推理速度优势的同时，在多种下游任务中一致超越现有预训练模型。

## 2. 模型架构


### 2.1 整体架构

```text
基因表达谱                            细胞-细胞关系图
    │                                      │
    ▼                                      ▼
基因编码器 (Gene Encoder)            图编码器 (Graph Encoder)
    │                                      │
    └──────────────┬───────────────────────┘
                   ▼
           融合表示 (Fusion)
                   │
          ┌────────┴────────┐
          ▼                 ▼
     基因预测头         细胞分类头
```

### 2.2 双编码器设计

- **基因编码器**: 标准 Transformer Encoder，处理基因表达序列（类似 Geneformer 的排序方案）
- **细胞关系编码器**: GNN（图神经网络），处理基于表达相似性构建的 kNN 细胞图
- 两者输出通过交叉注意力融合

### 2.3 关键创新点

预训练时不仅预测掩码的基因表达，还约束细胞嵌入在细胞关系图中保持局部一致性——即相似细胞的嵌入应该接近。



## 3. 核心创新


### 3.1 从"基因关系"到"细胞关系"的跨越

现有模型（Geneformer、scGPT）只建模基因-基因关系，CellPLM 首次将细胞-细胞关系引入预训练。

### 3.2 百倍推理速度

由于设计了高效的图编码器，推理速度比同期的 Transformer-only 模型快约 100 倍。

### 3.3 与同类模型对比

| 维度 | CellPLM | Geneformer | scGPT |
|------|---------|-----------|-------|
| 建模关系 | 基因 + **细胞** | 仅基因 | 仅基因 |
| 推理速度 | ⚡ 100x 更快 | 标准 | 标准 |
| 架构 | Transformer + GNN | Transformer | Transformer |
| 预训练任务 | 掩码预测 + 图对比 | 掩码预测 | 生成式 |



## 4. 数据预处理


标准流程：对数归一化 → 高变基因选择 → 构建细胞-细胞 kNN 图。



## 5. Tokenization 与输入编码


基因编码采用与 Geneformer 类似的排序 tokenization，基因按表达量降序排列。

细胞关系图基于表达谱的余弦相似度构建 kNN 图（k=15~30）。



## 6. 预训练


- **数据**: 约 3000 万细胞，涵盖多种人体组织
- **目标 1**: 掩码基因表达预测（类似 MLM）
- **目标 2**: 细胞关系对比学习（拉近邻接细胞嵌入）
- 双目标联合训练



## 7. 下游任务


| 任务 | 方法 | 性能 |
|------|------|------|
| 细胞类型注释 | 零样本 / Fine-tune | 一致最优 |
| 批次校正 | 嵌入 + Harmony | 竞争力强 |
| 基因网络推断 | 注意力权重 | 高质量 |

## 9. 关键概念 Q&A


**Q: CellPLM 和 Geneformer 的核心区别？**
A: Geneformer 是基因层面的（基因→token 排序→Transformer），CellPLM 在此基础上增加了细胞关系图，显式建模"哪些细胞相似"。

**Q: 推理速度为什么快？**
A: 图编码器使用高效的 GNN 而非全连接注意力，计算复杂度从 O(N²) 降至 O(Nk)（k 为近邻数）。
