## 2. 模型架构

### 2.1 评估维度

- **GRN 推断**: 模型嵌入是否支持基因调控网络重建
- **扰动预测**: 基因扰动后的表达变化预测
- **通路分析**: 通路激活/抑制的编码能力
- **跨物种泛化**: 模型是否跨物种适用

### 2.2 测试模型

scGPT、Geneformer、scFoundation、UCE、CellPLM 等

## 3. 核心创新

### 3.1 模型能力图谱

- 不同模型在不同生物任务上各有所长
- 没有模型在所有任务上全面占优
- 模型大小与生物任务性能不完全相关

### 3.2 启示

- 评估应向生物相关任务倾斜
- 任务特定的模型选择是必要的
- 需要更全面的评估标准
