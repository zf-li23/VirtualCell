## 2. 模型架构

### 2.1 漂移群岛框架

作者构造了已知最优嵌入的合成数据——漂移群岛：细胞群体在嵌入空间中形成分离良好的"岛屿"。通过系统地引入不同类型的退化（漂移），可以验证评估指标是否能正确识别最优嵌入。

### 2.2 两个核心仓库

- **Islander**: 官方实现，生成合成数据和评估指标。([GitHub](https://github.com/wanghanchen/Islander))
- **scib-metrics** (YosefLab): 标准的单细胞集成评估指标库。([GitHub](https://github.com/YosefLab/scib-metrics))

## 3. 核心创新

### 3.1 指标失败案例

| 指标类型 | 常见指标 | 失败表现 |
|---------|---------|---------|
| 批次校正 | kBET, iLISI | 对明显过度校正给出高分 |
| 生物保存 | NMI, ARI | 聚类性能与嵌入质量脱钩 |
| kNN 指标 | ASW, cLISI | 对退化嵌入不敏感 |

### 3.2 启示

- 现有指标系统存在系统性偏差
- 需要更严格的评估框架
- 合成数据验证应是标准做法
