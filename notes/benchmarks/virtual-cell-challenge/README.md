# Virtual Cell Challenge 学习笔记

> Virtual Cell Challenge（虚拟细胞挑战赛）是由 Arc Institute 和 CZ Biohub 联合发起的**首个系统化虚拟细胞基准测试**，被称为"虚拟细胞领域的 ImageNet 时刻"。它像图灵测试评估 AI 一样评估虚拟细胞模型——给定对照细胞的表达谱和扰动条件，模型能否准确预测扰动后的细胞状态？首届挑战（2025）吸引了 **4,325 个团队**参与，重点测试**跨上下文（跨细胞类型）泛化**能力。

### 核心思想

> **就像 ImageNet 推动了计算机视觉一样，一个标准化的、公共的基准可以推动虚拟细胞领域的发展。** 如果模型能准确预测细胞对扰动的响应——特别是在从未见过的细胞类型上——那它就通过了"虚拟细胞图灵测试"。

## 2. 挑战设计

### 2.1 2025 挑战：上下文泛化

首届挑战聚焦于**在未见过的细胞类型上预测扰动效应**：

- **训练集**: 150 个基因的 CRISPRi Perturb-seq 数据
- **验证集**: 50 个基因
- **测试集**: 100 个基因
- **目标细胞**: H1 人胚胎干细胞系（在训练阶段完全不可见）

### 2.2 评估指标

| 指标 | 说明 |
|------|------|
| **DE Score** | 差异表达基因的预测精度 |
| **Perturbation Distinguishability** | 按与真实扰动的相似度排序 |
| **MAE** | 总体预测平均绝对误差 |
| **综合评分** | 上述指标的加权组合 |

## 3. 核心发现

- 最佳模型已经能较好预测**同分布**的扰动，但在**跨细胞类型泛化**上仍有显著提升空间
- 模型集成（Ensemble）策略普遍优于单模型
- 预训练基础模型（如 scGPT、Geneformer）比从头训练的模型表现更好
- 挑战将成为年度系列活动，持续推动虚拟细胞领域发展
