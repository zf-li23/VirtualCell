## 2. 模型架构

### 2.1 测试任务

- **跨平台泛化**: 训练于 10x，测试于 Smart-seq2 等
- **跨组织泛化**: 训练于组织 A，测试于组织 B
- **新细胞类型**: 测试集中包含训练未见的细胞类型
- **批次效应鲁棒性**: 同一组织不同批次的泛化

### 2.2 对比模型

scGPT、Geneformer、scFoundation、UCE 等主流基础模型

## 3. 核心创新

### 3.1 主要缺陷

- **零样本性能显著低于有监督基线**: 微调后的模型远好于零样本
- **批次效应严重**: 模型对技术批次非常敏感
- **领域漂移**: 跨组织/跨平台时性能急剧下降

### 3.2 启示

- 零样本迁移目前不可靠
- 微调仍然是必要的
- 需要更好的预训练策略和评估标准
