## 2. 模型架构

### 2.1 Transformer 架构基础

论文首先回顾了 Transformer 的核心组件：
- **自注意力机制**: 建模输入序列中所有位置之间的关系
- **多头注意力**: 并行学习不同的关系模式
- **位置编码**: 为无序的基因表达数据提供位置信息
- **前馈网络**: 非线性变换层

### 2.2 单细胞适配策略

由于 scRNA-seq 数据缺乏天然的序列顺序，主要的适配策略包括：
- **基因级编码**: 将每个基因视为一个 token（如 scGPT, Geneformer）
- **表达值编码**: 将表达离散化为分箱 token
- **Pair 编码**: (基因, 表达值) 联合编码

## 3. 核心创新

### 3.1 全面涵盖的模型

| 模型 | 架构特点 | 预训练策略 |
|------|---------|----------|
| scBERT | 基于 BERT | 掩码基因预测 |
| Geneformer | 基于 BERT | 基因排名编码 |
| scGPT | 基于 GPT | 生成式预训练 |
| scFoundation | 基于 Transformer | xTrimo 架构 |
| CellPLM | 图+Transformer | 细胞间关系 |

### 3.2 关键见解

- Transformer 的泛化能力在单细胞领域具有巨大潜力
- 数据无序性是需要解决的核心挑战
- 当前的评估标准需要更严格的基准
- 结合生物学先验知识是重要的未来方向
