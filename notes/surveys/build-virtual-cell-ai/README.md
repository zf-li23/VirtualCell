## 2. 模型架构

### 2.1 虚拟细胞的概念

虚拟细胞（Virtual Cell）是指能够定量建模细胞状态与功能关系，并预测扰动后果的计算模型。

### 2.2 四个优先领域

论文提出建设虚拟细胞的四大优先级：

1. **数据基础设施**: 构建大规模、高质量的扰动数据集
2. **模型架构**: 开发能够整合多模态数据的基础模型
3. **评估标准**: 建立统一的基准和评估框架
4. **社区建设**: 促进跨学科协作和开放科学

## 3. 核心创新

### 3.1 纲领性意义

本文是虚拟细胞领域的纲领性文献，由多位领域领军人物联合撰写。

### 3.2 关键建议

- 需要像 HGP（人类基因组计划）一样的协作努力
- 标准化评估是关键瓶颈
- 多尺度建模（分子→细胞→组织）是终极目标

### 3.3 后续影响

直接催生了 Arc Institute 的 Virtual Cell Challenge（虚拟细胞挑战赛，Cell 2025）。
