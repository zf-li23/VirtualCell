### 核心思想

> CellTok 是一个早融合（early-fusion）多模态大语言模型，专为单细胞转录组学设计。通过基于 token 的细胞统一表示，支持细胞级别任务、群体级别任务和交互式问答。

---

## 2. 模型架构

### 2.1 早融合策略

在模型早期就将基因表达 token 和文本 token 融合，而非后期对齐。

### 2.2 统一 Token 表示

细胞被编码为一系列 token，可以与自然语言 token 进行交互。

## 3. 核心创新

### 3.1 早融合多模态

不同于 Geneformer/scGPT 等纯表达模型，CellTok 直接从输入端融合表达和文本。

### 3.2 交互式问答

支持自然语言与细胞数据的交互查询。
