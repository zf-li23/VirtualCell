# VCWorld 学习笔记

> VCWorld 是一个细胞级别的白盒模拟器（white-box simulator），将结构化生物学知识与 LLM 推理相结合，以可解释、数据高效的方式预测细胞对扰动的响应。核心思想是通过机制推理（mechanism reasoning）而非黑盒预测来理解细胞行为。包含 GeneTAK 基准数据集。

### 核心思想

> VCWorld 是一个细胞级别的白盒模拟器（white-box simulator），将结构化生物学知识与 LLM 推理相结合，以可解释、数据高效的方式预测细胞对扰动的响应。核心思想是通过机制推理（mechanism reasoning）而非黑盒预测来理解细胞行为。包含 GeneTAK 基准数据集。

## 2. 模型架构

### 2.1 三阶段管道

VCWorld 的推理管道分三个阶段：

1. **知识集成（Knowledge Integration）**: 从公开数据库构建开放世界生物学知识图谱（通路、PPI、基因调控等）
2. **证据检索（Evidence Retrieval）**: 使用语义和图感知相似度找到支持案例
3. **结构化推理（Structured Reasoning）**: 综合证据预测 DE/DIR，附带机制解释

### 2.2 白盒设计

不同于黑盒深度学习模型，VCWorld 的推理过程完全可解释，每一步都有明确的生物学依据。

## 3. 核心创新

### 3.1 白盒模拟器

与主流黑盒模型不同，VCWorld 提供完全可解释的推理链。

### 3.2 知识图谱驱动

利用公开生物学知识图谱，减少对大规模标注数据的依赖。

### 3.3 GeneTAK 基准

基于 Tahoe-100M 构建的基因级别扰动响应基准数据集。

## 9. 关键概念 Q&A

**Q: 白盒和黑盒的区别？**
A: 白盒模型的每个预测都有明确的生物学推理过程，黑盒模型只输出结果。

**Q: 需要 GPU 吗？**
A: 取决于使用的 LLM。本地 HF 模型需要，API 模型不需要。
