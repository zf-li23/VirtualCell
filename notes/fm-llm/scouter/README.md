### 核心思想

> Scouter 是一个架构简单的深度神经网络，用于预测未见基因的转录扰动响应。核心创新在于使用 LLM 从基因文本描述生成的嵌入作为模型输入，从而实现对未在训练中出现的基因的预测能力。通过 pip 可直接安装使用。

## 2. 模型架构

### 2.1 网络结构

Scouter 采用简单的深度神经网络架构：

1. **输入层**: 基因文本描述的 LLM 嵌入 + 基因独热编码
2. **隐藏层**: 若干全连接层
3. **输出层**: 预测扰动后的表达变化

### 2.2 零样本泛化

由于输入是 LLM 嵌入（而非基因 ID），模型可以通过已知基因的语义相似性推广到未见基因。

## 3. 核心创新

### 3.1 未见基因预测

利用 LLM 嵌入的语义泛化能力，首次实现训练时未见基因的扰动预测。

### 3.2 极简架构

证明了简单架构 + 高质量嵌入可以达到甚至超越复杂模型的效果。
