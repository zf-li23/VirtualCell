> Tabula 探索了将表格数据自监督学习范式应用于单细胞 RNA-seq 数据的可能性。核心思路是将 scRNA-seq 表达矩阵视为表格数据，利用表格数据的自监督预训练方法来学习细胞表示。相关工作显示，简单的方法（如 PCA）在某些场景下可以不亚于复杂的深度模型。

## 2. 模型架构

### 2.1 表格数据视角

将 scRNA-seq 的表达矩阵视为标准的表格数据（行=细胞，列=基因）。

### 2.2 SSL 策略

使用表格数据专用的自监督学习策略（如掩码特征重建、对比学习等）来预训练表示。

## 3. 核心创新

### 3.1 跨领域迁移

将表格数据领域的 SSL 成功经验迁移到单细胞分析。

### 3.2 简化假设

不依赖生物学先验（如基因关系、PPI 等），仅从表达模式中学习。

## 4. 数据预处理

### 4.1 标准预处理

对数归一化、高变基因筛选等标准的 Scanpy 流程。

## 5. Tokenization 与输入编码

### 5.1 数值输入

直接使用归一化后的数值表达值，无需离散化。
