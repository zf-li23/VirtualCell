# scNET 学习笔记

> scNET 是一个通过将单细胞基因表达数据与蛋白质-蛋白质相互作用（PPI）网络整合来学习上下文特异性基因和细胞嵌入的新方法。它采用基于 GNN 的双图架构，联合表示基因表达和 PPI 网络数据，利用注意力机制建模上下文中的基因-基因关系并细化细胞-细胞关系。最终产出的基因嵌入可用于基因功能预测、细胞状态发现等任务。

### 核心思想

> scNET 是一个通过将单细胞基因表达数据与蛋白质-蛋白质相互作用（PPI）网络整合来学习上下文特异性基因和细胞嵌入的新方法。它采用基于 GNN 的双图架构，联合表示基因表达和 PPI 网络数据，利用注意力机制建模上下文中的基因-基因关系并细化细胞-细胞关系。最终产出的基因嵌入可用于基因功能预测、细胞状态发现等任务。

## 2. 模型架构

### 2.1 双图架构（Bi-Graph Architecture）

scNET 同时构建两个图：

1. **细胞-细胞 kNN 图**: 基于表达相似性构建细胞关系图
2. **基因-基因 PPI 图**: 基于 BioGrid 数据库的蛋白互作网络

### 2.2 消息传递机制

- **基因图路径**: PPI 上的消息传递更新基因嵌入，融入已知的蛋白互作信息
- **细胞图路径**: kNN 图上的注意力机制细化细胞嵌入
- **联合更新**: 基因嵌入和细胞嵌入交替更新，相互增强

### 2.3 默认配置

- INTER_DIM=250（中间维度）
- EMBEDDING_DIM=75（嵌入维度）
- 最大训练轮数: 250
- 迷你批次: 3

## 3. 核心创新

### 3.1 基因表达 + PPI 融合

首次将单细胞表达数据和 PPI 网络通过双图架构联合学习。

### 3.2 上下文特异性建模

基因-基因关系不是固定的，而是根据细胞状态（上下文）动态调整。

### 3.3 基因嵌入的实用性

学到的基因嵌入可直接用于基因功能预测、共表达模块发现等。

## 4. 数据预处理

### 4.1 数据预处理

- 加载 AnnData 对象
- 可选：标准 Scanpy 预处理
- 基因名大小写处理（human_flag 控制）
- PPI 网络从 BioGrid 加载

### 4.2 图构建

- 细胞图: kNN（默认 k=15）
- 基因图: BioGrid PPI 子网

## 5. Tokenization 与输入编码

### 5.1 输入格式

- 原始输入: 基因表达矩阵（cell x gene）
- PPI 边: 从 BioGrid 预处理的边列表
- 无显式 tokenization，直接使用数值表达值

## 6. 预训练

### 6.1 训练目标

重建基因表达矩阵 + 图结构保持。

### 6.2 优化

- 优化器: Adam
- 最大 epoch: 250
- 早停策略
- 分批训练支持大规模数据

## 9. 关键概念 Q&A

**Q: scNET 和传统 GNN 的区别？**
A: scNET 同时使用双图（细胞图 + 基因图），而传统 GNN 通常只在单一图上操作。

**Q: PPI 网络的作用？**
A: 提供基因-基因关系的先验知识，帮助模型区分噪声表达信号和有生物学意义的共表达。
